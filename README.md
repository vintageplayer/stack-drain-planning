# Stack Drain Infra Debt POC

Monorepo for planning and a single-page web app that analyzes infrastructure lifecycle debt.

## Structure

- `apps/web`: Next.js + Tailwind single-page Infra Debt analyzer
- `docs/planning`: workflow, tasks, and planning templates

## Quick Start

```bash
cd apps/web
npm install
npm run dev
```

## Test

```bash
cd apps/web
npm run test
```

## Build

```bash
cd apps/web
npm run build
```

## Vercel

Deploy as a monorepo project with root directory set to `apps/web`.
